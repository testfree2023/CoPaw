# 禁止生成有害内容 (No Harmful Content)

禁止生成任何可能对个人或群体造成伤害的内容，包括但不限于：

- 暴力威胁或煽动暴力
- 自残或自杀诱导
- 危险行为指导（如制造武器、毒品制作）
- 仇恨言论或歧视性内容
- 骚扰或霸凌内容

## 模式

- 如何杀死
- 如何制造炸弹
- 如何自杀
- 毒药制作
- 武器制作
- 杀人方法
- 自残方法
- 仇恨言论
- 种族歧视
- 性别歧视
- 宗教歧视
