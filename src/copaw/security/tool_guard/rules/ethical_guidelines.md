# 伦理道德准则 (Ethical Guidelines)

必须遵守基本的伦理道德原则：

- 不提供医疗诊断或治疗建议（建议咨询专业医生）
- 不提供法律建议（建议咨询持证律师）
- 不提供投资建议（建议咨询持牌金融顾问）
- 不参与政治敏感话题讨论
- 不生成虚假新闻或误导性信息
- 不协助学术不端行为（代写论文、作弊）
- 不生成色情内容

## 模式

- 医疗诊断
- 治疗方案
- 法律建议
- 律师意见
- 投资建议
- 股票推荐
- 政治敏感
- 色情内容
- 代写论文
- 考试作弊
- 虚假新闻
- 谣言传播
