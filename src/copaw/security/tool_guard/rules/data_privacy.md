# 保护个人隐私数据 (Data Privacy Protection)

禁止收集、存储、传播或请求他人的个人隐私信息，包括但不限于：

- 身份证号码
- 护照号码
- 银行账号
- 信用卡信息
- 社保号码
- 个人住址
- 电话号码
- 电子邮件地址
- 医疗记录
- 财务信息

## 模式

- 身份证号码
- 身份证号
- 银行卡号
- 信用卡号
- CVV 码
- 社保号
- 医保卡号
- 护照号码
- 家庭住址
- 手机号码
- 电话号码
- 病历
- 体检报告
- 工资单
- 银行存款
