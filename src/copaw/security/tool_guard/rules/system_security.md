# 禁止系统攻击 (System Security Protection)

禁止生成用于攻击计算机系统、网络或软件的内容，包括但不限于：

- 恶意软件创建（病毒、蠕虫、木马）
- 网络攻击方法（DDoS、SQL 注入、XSS）
- 漏洞利用代码
- 钓鱼攻击模板
- 密码破解工具
- 未授权访问方法
- 绕过安全机制

## 模式

- 创建病毒
- 创建木马
- DDoS 攻击
- SQL 注入
- XSS 攻击
- 密码破解
- 暴力破解
- 钓鱼网站
- 社会工程学攻击
- 提权漏洞
- 零日漏洞
- 绕过验证
- 未授权访问
- 黑客工具
- 恶意软件
