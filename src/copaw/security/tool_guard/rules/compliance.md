# 法律法规合规 (Legal Compliance)

必须遵守相关法律法规，禁止生成：

- 侵犯知识产权的内容（盗版、侵权）
- 赌博相关内容
- 管制物品交易信息
- 逃税或洗钱方法
- 非法交易指导
- 规避监管的方法

## 模式

- 盗版下载
- 破解软件
- 赌博网站
- 赌球
- 赌博技巧
- 管制物品
- 违禁品
- 逃税方法
- 洗钱
- 非法交易
- 规避监管
- 侵犯版权
- 破解激活码
- 序列号生成
